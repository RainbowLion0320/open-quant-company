# Maintainers

## Current Maintainer

- `@RainbowLion0320` — project owner and release maintainer

## Maintainer Responsibilities

Maintainers are expected to:

- Keep the default branch releasable.
- Review pull requests for architecture, security, data integrity, and documentation impact.
- Keep community standards and security policy current.
- Triage issues and label them by area.
- Coordinate releases and changelog updates.
- Remove secrets or sensitive local data if they are accidentally submitted.

## Becoming a Maintainer

Additional maintainers may be added after repeated high-quality contributions and demonstrated care for the project's data, research, execution, and open source boundaries.
