# 开发计划

> 更新: 2026-05-23

`docs/development-plan.md` 是当前活跃计划入口。本目录用于存放专题未来计划和已完成计划归档。

## 规则

- `docs/development-plan.md` 只保留一个总体活跃计划。
- 尚未启动的专题未来工作流放在 `docs/plans/<topic>.md`。
- 完成或被取代的计划移动到 `docs/plans/archive/`。
- 不把 PRD/spec/wiki 页面当作 sprint 清单使用。
